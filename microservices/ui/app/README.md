## Local Deployment

### Prerequisites

- [npm](https://www.npmjs.com/get-npm)

### Deploy

1. Download or clone the repo
2. ```cd microservices/ui/app```
3. Run ```npm install```
4. Run ```npm start``` 

Your app will be live on `localhost`
